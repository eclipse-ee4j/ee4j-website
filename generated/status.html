EE4J Project Bootstrapping

This page provides an overview of the bootstrapping state of the EE4J projects. This is based on information that is tracked by the Eclipse Foundation. Please report any discrepancies to emo@eclipse.org.

The above graph charts the following states:

  1. Open Source Project Proposal has been provided to the Eclipse Foundation

  2. Proposal is posted for review

  3. Trademark search initiated

  4. Trademark search resolved

  5. Open Source Project Created

  6. Provisioning requested

  7. Committers and resources provisioned

  8. Initial Contribution provided to the Eclipse IP Team

  9. Initial Contribution approved for Check-In

  10. Initial Contribution Pushed to Git Repository

  13. Project has engaged in its first Release Review

For more information, please see the Starting an Open Source Project section in the Eclipse Project Handbook.
